근거

아트북에서 추출한 '구도 문법(Composition Grammar)'

화조도(花鳥圖)와 Bassano의 〈The Way to Calvary〉 분석 페이지가 보여주는 것: 명작은 우연이 아니라 삼각구도·방향성 흐름·황금구조·초점 위계라는 반복 문법 위에 서 있다. 이 네 축이 AURA 구도 모델의 라벨 체계가 된다.

삼각구도
triangular mass
방향성 흐름
directional flow · S/대각선
황금구조
golden · 1/3 grid
초점 위계
focal hierarchy
1 — 경계
브라우저 vs 백엔드

브라우저(엣지)에 남김

  • 색 추출·팔레트 양자화 (F3)
  • 밝기·채도·색온도 아우라 (F2)
  • 휘도 무게중심·삼분할/황금 오버레이 (F1 기초)
  • 기본 색·톤 그레이딩 (F5 기초)
  • 이미지→파형→음악 (F6, Web Audio)
  • 프라이버시: 원본은 기기에서 우선 처리

ML 백엔드로 넘김

  • 구도 문법 인식 (삼각/흐름/황금/초점)
  • 명작 코퍼스 학습 → 미적 스코어링
  • 찍기 전 가이드 생성 (장면 인식 기반)
  • 프로급 재가공 (스타일·리라이팅)
  • 피사체 분리·재배치 제안
  • 미적 정체성 개인화 학습
2 — 백엔드 모듈
6개 모델 파이프라인
M1

구도 분석 (Composition Grammar)

Saliency + Segmentation + Depth

돌출맵(saliency)으로 초점, 분할(segmentation)로 피사체 덩어리, 깊이(depth)로 층위를 얻고 → 삼각구도 적합도·방향성 벡터·황금/삼분할 정합·초점 위계를 산출. 아트북 오버레이(삼각형·흐름 화살표)를 자동 생성.

in image → out {triangle_fit, flow_vectors[], golden_score, focal_rank[]}
M2

명작 코퍼스 & 미적 스코어링

Aesthetic model (NIMA-style) + fine-tune

동양 화조도·서양 고전(라벨된 구도 오버레이 포함)으로 파인튜닝한 미적 평가 모델. 사진에 0~10 미적 점수 + "왜"(어떤 문법을 지키거나 어겼는지) 설명을 부여.

in image → out {aesthetic_score, grammar_violations[], nearest_master[]}
M3

찍기 전 가이드 생성

VLM (scene→guideline)

실시간 장면/프리뷰를 보고 구도 문법에 맞춰 구체 가이드 생성: "피사체를 우측 1/3선으로", "가지의 대각선을 살려 삼각을 완성", "역광으로 실루엣". AR 오버레이용 좌표까지.

in live frame → out {guidelines[], overlay_coords[], target_grade}
M4

프로 재가공 (Enhance/Restyle)

Diffusion img2img + LUT + relight

미적 점수를 높이는 방향으로 색·톤·조명 재가공. 명작 스타일 LUT + 확산모델 img2img(약한 강도, 원본 보존) + 리라이팅(광원 방향 추정→보정). 항상 before/after로 반환.

in image + target_style → out {enhanced_image, edit_layers[], delta_report}
M5

피사체 재배치 제안

Segmentation + inpainting

초점 피사체를 분리해 황금/삼분할 위치로 옮긴 시안을 인페인팅으로 생성(제안용). 크롭 재구성도 함께.

in image + focal_mask → out {recompose_variants[]}
M6

미적 정체성 학습

Embedding + user profile

사용자 사진들의 색·구도·아우라 임베딩을 누적해 개인 취향 벡터를 학습. 코치·재가공·음악을 개인화. (Virtual ME 파동 정체성과 연결)

in user_library → out {taste_vector, palette_signature, aura_baseline}

모듈모델 후보단계

M2 미적NIMA/CLIP-aesthetic + 명작 파인튜닝MVP
M3 가이드경량 VLM (장면→텍스트+좌표)Scale
M4 재가공SDXL img2img(저강도)+LUT+relight netScale
M5 재배치SAM + LaMa inpaintingExpand
M6 정체성임베딩 축적 + 개인 프로파일Expand
